# Foodlist Express API with Swagger

An API documentation with Swagger Open API specification 3.0.

## Prerequisites

* Node.js

## How to start webserver

```
npm install
npm start
```

## APIs

APIs for list food and get by food id.

## API Docs

URL : http://localhost:3000/api-docs
